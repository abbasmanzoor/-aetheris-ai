# 📖 Overview

**Aetheris AI** is a modern, fully responsive, and production-ready **SaaS Landing Page** designed for a fictional AI technology company.

The project demonstrates modern frontend development practices, including responsive layouts, advanced animations, glassmorphism UI, dark/light mode, accessibility, and performance optimization.

It was developed as part of the **TEYZIX CORE Internship (June Batch)** for **Task FEWD-3**.

# ✨ Features

## 🎯 Core Sections

- 🚀 Hero Section with animated headline and CTA buttons
- 📦 Product showcase with interactive cards

# 🚀 Getting Started

Follow these steps to run the project locally.

## Prerequisites

Make sure you have installed:

- Node.js **v18 or higher**
- npm or Yarn

---

## Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/aetheris-ai.git

# Navigate to the project directory
cd aetheris-ai

# Install dependencies
npm install

# Start the development server
npm run dev
```

---

## Build for Production

```bash
npm run build
```

---

## Preview Production Build

```bash
npm run preview
```
